# Zhǐ (紙)

A light theme that aims for eyes comfort rather than heavily invested in
contrast.

## Installation

For packer: `use 'ingram1107/vim-zhi'`

For vim-plug: `Plug 'ingram1107/vim-zhi'`

## Usage

```viml
syntax on
colorscheme zhi
```
